# Suggested Folder Structure

```text
AI Brain/
  Shared/
    Profile.md
    Preferences.md
    Active Projects.md
    Decisions.md
    Open Loops.md

  Machines/
    Codex - Computer 1/
      Current Context.md
      Session Log.md
      Local Setup.md

    Codex - Computer 2/
      Current Context.md
      Session Log.md
      Local Setup.md

  Projects/
    Obsidian-AI-Brain/
      Overview.md
      Current State.md
      Decisions.md
      Next Steps.md
      Sessions/
        YYYY-MM-DD-session-title.md
```

Use one folder per computer under `Machines/`. Use one folder per important project under `Projects/`.

Templates live in `templates/` so they can be copied manually or through `scripts/brain.mjs`.
